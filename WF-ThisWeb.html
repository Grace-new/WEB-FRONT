HTML Basic

컨텐츠 구조 정의하는 마크업 언어 일련의 요소로 이루어져 있음.
* 요소 = <태그 속성> 컨텐츠 <태그>
* element = <tag attribute> contents <tag>

+ 속성 구조 < 요소이름 속성 = "" >
- 속성과 속성 사이에는 공백이 있어야 함
- 어떤 속성들은 내용을 갖지 않음

img 의 alt 속성 : 이미지 대체텍스트

CSS Basic

link href="styles/style.css" rel="stylesheet" type="text/css"
rel : 관계 설명, stylesheet, icon, etc
css ruleset 해부
선택자=요소이름 , 또다른 선택자 { 선언 ( 속성 : 속성값; ) }
  • #id
  • .class
  • img[src]
  • a:hover
  • p:first-child
    • : 첫번째 p에 적용
  • h1,h2,h3가 뭉탱이로 있으면 hgroup을 사용해주면 됨.

  • line-height : 줄의 높이
  • letter-spacing : 자간
  • padding : 안쪽 간격
  • margin : 바깥쪽 간격 ( 상/하단, 좌/우측 )

    • auto는 가능한 수평공간의 왼쪽과 오른쪽을 같게 나눠주는 특별한 값
  • text-shadow : 글자에 그림자 적용
  • border-padding : 바깥쪽에 놓인 실선
  • display : 요소의 표시 형식
